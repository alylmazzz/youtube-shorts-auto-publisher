🔐 Refresh token ile otomatik yenileme
Bu uygulama yalnızca senin bilgisayarında çalışacak şekilde tasarlandı. Client ID, Client Secret ve Refresh Token tarayıcıda değil, lokal Node server klasöründeki .youtube_oauth_local.json dosyasında saklanır. HTML her gerektiğinde lokal serverdan yeni access token ister.
Lokal refresh token sistemi aktif
https://www.googleapis.com/auth/drive.readonly https://www.googleapis.com/auth/youtube.upload https://www.googleapis.com/auth/youtube.force-ssl
Henüz lokal token server test edilmedi.
Tokenlar saat başı değiştiği için bu mod access token alanlarını otomatik doldurur. Refresh tokenı yalnızca senin bilgisayarındaki lokal server saklar. Kendi bilgisayarın dışında kullanma.
